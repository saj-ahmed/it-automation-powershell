# IT Automation PowerShell Toolkit

Infrastructure automation scripts for Microsoft-based enterprise environments.

## Overview

This repository contains practical PowerShell scripts designed for real-world IT administration tasks across Active Directory, Microsoft 365, and endpoint management environments.

These scripts reflect enterprise support activities such as user provisioning, reporting, device management, and operational automation.

---

## Use Cases

* Active Directory user management
* Microsoft 365 reporting
* Endpoint inventory collection
* System health checks
* Operational IT automation tasks

---

## Scripts Included

### 1. Active Directory User Creation

Automates bulk user creation from CSV input.

### 2. Microsoft 365 License Report

Exports user licensing information from M365 tenant.

### 3. Device Inventory Report

Collects system hardware/software information from endpoints.

### 4. Service Health Check

Checks Windows services and outputs status report.

### 5. Disk Space Monitoring

Identifies low disk space across servers/workstations.

---

## Requirements

* PowerShell 5.1+
* ActiveDirectory Module (for AD scripts)
* Microsoft Graph / MSOnline (for M365 scripts)
* Administrative permissions (where required)

## Purpose

This project demonstrates real-world IT automation experience in enterprise environments, aligned with Infrastructure Engineering and Modern Workplace roles.
